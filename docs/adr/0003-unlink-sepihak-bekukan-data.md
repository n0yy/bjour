# Unlink sepihak dengan pembekuan data bersama

Pemutusan Partnership bisa dilakukan sepihak dan berlaku seketika — tidak butuh persetujuan pihak lain, karena mekanisme dua-pihak memungkinkan mantan pasangan yang tidak kooperatif menyandera akses ke data finansial pihak lain. Saat Unlink, Shared Goal membeku menjadi arsip baca-saja di kedua akun (riwayat Contribution per pihak tetap tercatat sebagai bukti), dan akses transparansi dicabut seketika. Data bersama tidak dihapus, karena catatan kontribusi justru paling dibutuhkan saat berpisah.
