# Input transaksi manual-first, tanpa bank-sync di MVP

Meski BJour adalah money manager, MVP sengaja tidak mengintegrasikan bank-sync. Aggregator Indonesia (Brick, Finantier, Ayoconnect) berbiaya per-user sejak hari pertama dan mewarisi kerapuhan sinkronisasi yang menjadi keluhan utama user Finku — persis kelemahan kompetitor yang ingin kami hindari. Semua Transaction dan Contribution diinput manual, dibuat semulus mungkin; bank-sync dipertimbangkan ulang setelah produk terbukti.
